# diceGame

'Roll the dice' game app built with React.


### To clone the repo
```shell
$ git clone https://github.com/jen0828/diceGame.git
$ cd diceGame
```

### To run react app
``` shell
$ npm install
$ npm start 
# Start the server at localhost:3000
```
### To install cypress
```
npm install --save-dev cypress
```

### To run cypress test
```
npx cypress open
```
